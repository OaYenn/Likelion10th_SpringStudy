4) 클라이언트 -> 서버 요청 : @RequestBody
    서버 -> 클라이언트 응답 : @ResponseBody

* RDBMS
 : 컴퓨터에 정보를 저장하고 관리하는 기술
 : 매우 고도화된 엑셀이라고 생각하자!
* 세션에서 사용할 RDBMS : H2 / MySQL
* JPA
 : 자바로 DB를 사용하도록 도와주는, 일종의 '번역기'
 : 테이블 = Domain, SQL = Repository
* extends
 : 클래스의 상속
 : 이미 만들어둔 클래스를 가져다 씀.
   → 상속받은 클래스의 멤버변수와 메소드를
       고대로 물려 쓰게 됨.
* DTO(Data Transfer Object)
 : 완충재로 활용
 : <계층간 데이터 전달자 역할>
   클라이언트 요청에 포함된 데이터를 담아 서버 측에 전달,
   서버 측의 응답 데이터를 담아 클라이언트에 전달.
